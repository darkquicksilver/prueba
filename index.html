Implementación

Como se efectua la implementacion del componente, incluye la declaracion de assets, estructura HTML e inizialización en JS.

Declaración de assets del componente.

<!-- CSS Global -->
<link rel="stylesheet" href="global.css">

<!-- JS Global -->
<script src="global.js"></script>

Declaracíon del HTML del componente.

<div class="tim-componente" data-variable="valor1">
  <div> Contenido  </div>
  <div> Contenido  </div>
  <div> Contenido  </div>
  ...
</div>

Llamada del componente en JS

$(".tim-componente").componente();
